## 小米智驾撞车维权指南

完整文档：[小米智驾撞车维权指南.md](./小米智驾撞车维权指南.md)

**场景**：智驾/NOA 运行中撞墙角，全程无预警、无接管提示，小米拒赔要求进保。

### 核心结论

1. 手册写「墙角识别受限」≠ 小米可完全免责
2. L2 辅助驾驶下驾驶员仍是第一责任人，但 **零预警、不退出、不提示接管** 可主张警示缺陷
3. **暂不要先进保** — 易默认自行承担，削弱后续追偿
4. 维权顺序：**固定证据 → 申请数据 → 定损 → 再决定报交警/12315/诉讼**

### 立即行动

- 拍照/录像、导出行车记录仪、保存小米沟通记录
- 书面申请智驾数据提取与封存
- ❌ 暂不维修、暂不先进保

### 12315 四大重点

1. 「已知局限」≠「零预警也可免责」
2. 拒绝提供事故数据
3. 宣传是否误导
4. 售后推诿只让进保
